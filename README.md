## 📌 Executive Summary
This project uses Machine Learning and Generative AI to optimize college mess food planning. It predicts daily food demand based on attendance and generates intelligent weekly menus to reduce food wastage and improve efficiency.

---

## 🚀 Features
- 📊 Attendance vs Mess usage analysis
- 🤖 ML-based food demand prediction
- 🧠 AI-generated weekly menu planning
- 📉 Food wastage reduction insights
- 🌐 Flask web interface

---

## 🧠 Tech Stack
- Python (Flask)
- Pandas, NumPy
- Scikit-learn (ML model)
- OpenAI API (Generative AI)
- HTML, CSS

---

## ⚙️ How It Works
1. User enters attendance & food data
2. ML model predicts food demand
3. AI generates optimized menu
4. System shows insights via dashboard

---

## 🏆 Impact
- Reduces food wastage
- Improves mess planning efficiency
- Saves cost
- Improves student satisfaction
